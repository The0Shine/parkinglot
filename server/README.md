# Node js server boilerplate for esp8266

## Mô tả
Mẫu server nodejs cho esp8266, môn học cnpmn kma c6

## Yêu cầu hệ thống
- Node.js
- MongoDB
- npm
- Git

### 1. Clone dự án
```bash
git clone https://github.com/Horob1/node-js-server-for-esp8266
cd node-js-server-for-esp8266
npm i
npm run dev
````
